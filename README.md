# ⚡ Neon Proxy Hub

A cyber-themed browser-style UI with:
- Tabs
- URL navigation bar
- Back/forward history
- Full-screen iframe browser window

## 🚀 How to run locally
Just open `index.html` in your browser.

## 🌐 Deploy on GitHub Pages
1. Upload this repo to GitHub
2. Go to Settings → Pages
3. Select branch: `main`
4. Click Save
5. Your site will be live at:
   `https://yourusername.github.io/neon-proxy-hub/`

## ⚠️ Notes
Some websites may not load in the iframe due to security restrictions (X-Frame-Options).
